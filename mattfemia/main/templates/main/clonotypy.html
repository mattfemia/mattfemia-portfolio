Clonotypy¶

Clonotypy is a package for analyzing single cell immune profile data.¶

The primary purpose of Clonotypy is to provide easy access to complex analysis of scVDJ-seq runs. It has support for comparing several scVDJ-seq samples from various pre-processing pipelines, descriptive and multivariate statistics, multi-omic analysis, advanced visualization, and API integrations to many common immunology databases.

This notebook was created to demonstrate basic functions by showing descriptive analysis functions used on a single sample of scTCR-seq data.

CDR3aa-Composition¶

Clonotypy has several functions related to the amino acid composition off CDR3 chains. These functions are primarily focused on calculating position weight matrices (PWM), but can also be represented as position frequency (PFM) or probability matrices (PPM) as well.

The position matrices can each be used to plot heatmaps or seqlogos for a given amino acid chain length to further characterize the chemical composition.

Repseq objects can be converted to position matrices via the to_PositionMatrix() method which has the flexibility to specify matrix_type(pfm, ppm, pwm), cdr3_length, cdr3(aa or nt, for amino acid or nucleotide), chain(receptor chain), pseudocount, and more.

In [ ]:
# Position Frequency Matrix
pfm = clones.to_PositionMatrix(matrix_type='pfm', 
                               length=15, 
                               cdr3='aa', 
                               chain='TRA', 
                               pseudocount=1, 
                               col_prefix='p')
pfm
In [ ]:
pfm.plot_matrix(palette='immunovue', annot=True, linecolor="#ffffff", linewidths=1)
In [ ]:
pfm.plot_seqlogo(palette='immunovue', alpha=1)

Easily convert between different position matrices:

In [ ]:
clp.composition.pfm_to_ppm(pd.DataFrame(pfm))

Beyond-Basics¶

Clonotypy is intended to help investigate multiple sequencing runs, or replicates. Therefore, it's primary functionality extends beyond the basic descriptive statistics provided in this tutorial.
